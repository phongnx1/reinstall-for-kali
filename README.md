# Install for Kali
## [Adding a new user in Kali Linux](https://www.linkedin.com/pulse/20140502074357-79939846-adding-a-new-user-in-kali-linux)

- useradd -m username

- passwd username

- usermod -a -G sudo username

- chsh -s /bin/bash username
